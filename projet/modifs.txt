Petit fichier pour te dire ce que j'ai modifié dans bison.

- J'ai rajouté plusieurs regles que tu avais oublié:
  	   Un content peut être de la forme :
	        |   LABEL container content
	   pour accepter des choses telles que :
	   { div{"content"} div{"content"} }
	   ce qui donne :
	      { container }
		       { LABEL container content }
			       { LABEL container LABEL container content }
				       { LABEL container LABEL container empty }

		Un content peut aussi être de la forme :
		   	| LABEL '/'
		pour accepter des choses telles que :
		{ foo/ }

- J'ai rajouté des commentaires significatifs pour le débuggage.
  	   (un peu lourd a l'écran mais ca permet de bien identifié ce qu'il se passe)
